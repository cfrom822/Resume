HTML DOM Elements with Javascript

In order to manipulate an HTML element we must first find it and in Javascript we can do this in many different ways.

By id

The easiest and most commonly used method is getElementById().
For example if we have the following:
<p id="test">Hello</p>
We can use:
let testVar = document.getElementById('test');

By tag name

We can also get all elements based on their tag using the method getElementsByTagName().
For example if we have the following:
<h1>Test</h1>
We can return an array of the elements with:
let testVar = document.getElementsByTagName('h1');
Or if we want to target only one we can select it based on the placement in the array:
let testVar = document.getElementsByTagName('h1')[0];

By class name

One more method is to get elements by class name using getElementsByClassName().
For example if we have the following:
<div class="test"></div>
We can get an array of the elements with:
let testVar = document.getElementsByClassName('test');
Or simply one element in the array with:
let testVar = document.getElementsByClassName('test')[0];

String methods in Javascript

There are a wide variety of methods for use with strings in Javascript. The following are a few of the examples.

String Length

We can easily identify the length of a string using the length property.
For example if we have the following:
let str = "This is a test";
We can use:
let length = str.length;
To which our length variable will receive the number 14.

Slice

We can also obtain a part of a string and place it into a new variable by using slice().
For example if we have the following:
let txt = "Strawberry, Grape, Banana";
We can use:
let berry = txt.slice(0,10);
To which we recieve 'Strawberry'.
We can also go from the end using negative numbers
let last = txt.slice(-6);
To which we receive 'banana'.

Converting to upper and lower case

One more method is to convert a string to a different casing using toUpperCase() and toLowerCase().
For example if we have the following:
let name = "First Name";
We can make it all uppercase using:
let nameUpper = name.toUpperCase();
To which we receive 'FIRST NAME'.
Or we can make it all lowercase using:
let nameLower = name.toLowerCase();
To which we receive 'first name'.

Printing Text with Javascript

There are a number of ways we can display text with Javascript. On the web page itself, in the console or as a notification prompt.

Printing Text to the Web Page

In order to print some text directly on to the webpage we can use innerHTML.
For example if we have the following:
<p id="test"></p>
We can use the document.getElementById(id) method to find the element and then change the content with innerHTML:
document.getElementById('test').innerHTML = "This is a test";

Printing Text to the Console

We can also print text directly to the console which can help us debug our code using the console.log() method.
We can use the below:
console.log("This is a test");
To which we would see "This is a test" in our console.

Printing to an Alert Box

One more method is to create an alert box containing our text with alert().
By using the below:
alert("Hello World!");
We receive an alert box with the content 'Hello World!'.

NOTE

We can also see the output of our variable by using them as replacement inside our brackets.
For example if we have:
var x = 5+7;
We can use any of the following:
document.getElementById('test').innerHTML = var;
To which our paragraph text changes to 12.
console.log(x);
Here we receive 12 in the console.
alert(x);
Finally we get a prompt with the content 12.

Variable and Constant Declarations

In Javascript we have 3 different ways of declaring a variable, var, let and const.

Var

The first method is to use var.
For example:
var x = 5;
var y = "This is a test.";
var z = x + 7;

Let

The next method is to use let.
For example:
let x = 7;
var y = "Hello World!";
var z = x + 13;

Const

The final method is to use const.
PLEASE NOTE: A variable declared with const CANNOT be redeclared!
For example:
const PI = 3.1416;
const str = "Cannot change.";
const z = 5*7;

Functions in Javascript

In Javascript we use functions to execute a block of code to perform a task for us such as performing a mathematical operation.

Creating a Function

The usual layout of a function can be seen below:
function functionName(parameter1, parameter2){
 // your code here
}
We first use the function keyword, we then name our function, put any parameters we would like to include inside our parentheses and finally our code to be executed inside our curly brackets.

For example:
let x = 7;
var y = 5;
function sum(x, y){
 return x + y;
}

Invoking a function

When we use () we access the function immediately and get a result. When we access a function without () we instead get the function object.
For example in regard to our sum function above, stating sum will give us:
Æ’ sum(x, y){
 return x + y;
}
But if we give sum(x, y) we instead get:
3

For Loops

In Javascript we have 3 different ways of creating a for loop.

  • For
  • For in
  • For of

For

The first way is to use the for loop which will loop through a block of code a number of times.
For example:
var x = 0;
for(let i = 0; i < 10; i++){
 x = x + 2;
}

For in

The next method is to use for in which will loop through the properties of an object or places in an array.
For example:
let person = {first:"John", last:"Doe", age:25, gender:"male"};
for(let i in person){
 console.log(person[i]);
}

For of

The final method is to use for of. This will loop over the values of an object or an array.
For example:
var y = ["Grape", "Banana", "Orange", 1, 2, 3];
for(let i of y){
 console.log(i);
}

Objects

In Javascript we have objects which are similar to variables only objects hold many values.

Defining an Object

To define an object in Javascript we first use a variable declaration, followed by a variable name, however when we declare our values we place them inside of curly brackets {} and use key:value pairs or properties as seen below.
Example:
var car = {make:"Toyota", model:"Corolla", year:2021, color:"Blue"};

Accessing Object Properties

In order to access an object property we can declare the object name, then a period, followed by the property name.
For example:
car.make+" "+car.model;
Will gives us Toyota Corolla.

Type Conversion in Javascript

We can use Javascript to change the data type of our variables and convert them into a new variable.

Number

In Javascript we have a few ways we can convert something to a number. We can use Number(), parseInt() and parseFloat().
For example we can do any of the following:
Number(true);
parseInt("5");
parseFloat("3.14");

String

We can also convert numbers to strings using the method String().
Some examples include:
String(x);
String(5);
String(5+7);

Comparisons in Javascript

In Javascript we use comparison operators to check for true and false values.

Operators

The operators of Javascript include equal to ==, not equal !=, greater than >, less than << /strong>, greater than or equal to >= and less than equal to <=< /strong>.
Examples of the above:
x == 5;
y != 7;
x > y;
y < x;
x >= y;
y <= x;

Checking for type

In Javascript we also have the ability to check if the data type is the same or not with equal value and type === and not equal value or not equal type !==.
Structure remains the same but output can vary:
x === 5;
5 !== "5";
7 !== 3;

Dates

Using Javascript we can obtain the current date and time using the browsers time zone.

Creating a Date Object

Dates are objects and can be created using new Date() as shown below.
var d = new Date();
We can also specify the values instead of getting them from the browser. The order of the object is year, month, day, hour, minute, second and millisecond.
var d = new Date(2021, 12, 25, 23, 30, 0, 0);

Displaying Dates

In order to display a date we can use a few different methods depending on how much we would like to have displayed.
For example:
d.toString();
d.toDateString();
